# Bots

This repo is my collection of Discord bot experiments and utilities.

Most of the work here came from stuff I actually wanted in servers I was in: chore tracking, Canvas assignment reminders, music commands, moderation tools, and random API-based commands.

## Main pieces

| Folder | What it does |
|--------|--------------|
| `Chore Bot/` | Main Discord bot with chore tracking, Canvas reminders, SQLite-backed storage, logging, and utility commands. |
| `cogs/` | Modular command files for APIs, moderation, music, and error handling. |
| `Add-Ons and Utils/` | Extra helpers and side experiments. |
| `KTP website/` | Website-related files from another side project. |

## Features

- Discord slash/message command handling
- Chore assignment and tracking
- Canvas LMS assignment reminders
- SQLite storage for persistent bot data
- Music and moderation commands
- Logging and error handling utilities

## Stack

- Python
- discord.py / Pycord experiments
- SQLite
- Canvas API

## Notes

This is more of a working repo than a single polished app. I use it to build useful Discord tools, test bot ideas, and keep improving how I structure Python projects.
